# SALC WebLearn
WebLearn is a responsive distance learning website developed for the Springfield Adult Learning Center (SALC). This design utilizes a versatile mobile-first design with PHP, HTML5, CSS3, jQuery, SASS, Grunt, and Bootstrap. Encapsulated in WebLearn are two academic learning labs. ABE Lab is designed for students pursuing High School Equivalency Test preparation classes. ESOL Lab is a resource to help recent immigrants at SALC learn English.
