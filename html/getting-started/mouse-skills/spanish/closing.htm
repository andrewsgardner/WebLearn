Cerrando Una Aplicación

Esta es la última página...
Un método de cerrar aplicaciones se hace por pulsando el equis en la esquina derecha de la ventana emergente. Esta acción cierra la ventana que visita ahora. Fíjese que la ventana mas grande también tiene el equis en la misma esquina.

Otra manera de cerrar aplicaciones es por arrastrando el cursor hacia el menú desplegable que se llama "File". File queda a la cumbre de la pantalla. Púlselo y selecciona el opción "Exit" para cerrarla. Así se puede cerrar todas las aplicaciones que están abiertas.
