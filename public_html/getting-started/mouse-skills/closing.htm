Closing an Application

Last Page...
You have already practiced one method for closing an application - clicking on the in the top right-hand corner of the pop-up windows on the previous pages. This method closes the window you are currently viewing. Notice the larger windows also have the in the top right-hand corner.

Another way to close an application is to click on the word "File" at the top left of the screen and then click on "Exit" from the list that drops down. This method uses the skills you learned when we talked about "drop down menus" earlier. This method closes all open windows of an application.

Congratulations!
