Using the Scroll Box

Notice within the scroll bar that there is a box that stands out from the rest of the bar. This box varies in size depending on how much additional information there is beneath the visible part of the screen.

To scroll up or down the page using this box, position the cursor over the scroll box, hold down the left mouse button and drag the scroll box up or down. Release the mouse button when you have reached the desired location. This method uses the clicking and dragging technique you learned earlier. Try this now.
