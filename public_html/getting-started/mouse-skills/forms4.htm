  • The small circles in the list to the left are radio buttons.

  • Radio buttons allow you to select one item from a list of options.

  • Select one of the options by clicking in the circle next to it.

  • Notice that a dot will appear in the circle of the item you selected.

  • If you select a second option, the previous option will automatically be "unselected".
