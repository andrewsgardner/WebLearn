Utilizando El Botón Derecho

 

El botón derecho abre lo que se llama el menú de opciones o 'context menu.' Mire el ejemplo a la derecha. Este menú le permite realizar las operaciones comunes que aparecen en la barra de menú a la cumbre de la pantalla. Trate de oprimir el botón derecho ahora para abrir el menú de opciones y para cerrarlo arrastra el cursor afuera del menú y pulse el botón izquierdo. Así se hace si por casualidad lo abre sin querer. Atención: Algunas bibliotecas han inutilizado el botón derecho de las computadora, así que puede ser que este rasgo no funcionara.
