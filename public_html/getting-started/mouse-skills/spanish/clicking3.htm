Pulsar, Arrastrar y Soltar El Ratón

 

    El usuario puede empezar a arrastrar un objeto pulsando el botón del ratón y moviéndolo. Así también se puede marcar texto y realizar otras funciones. Para mover el cuadro rojo, coloque el cursor sobre la imagen. Ahora, oprime el botón izquierdo y mientras la tenga oprimido, arrastra el ratón a otra parte de esta página. Se termina de arrastrar cuando suelta el botón del ratón.

Practique con el cuadro rojo por un rato.
