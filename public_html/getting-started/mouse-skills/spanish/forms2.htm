Habilidades Adicionales: Menú Despegable

 

¿A donde quieres ir?
  • La caja a la izquierda con la palabra "escoge" se llama el menú despegable o drop down menu. Este menú se distingue de una lista en el hecho de que el usuario debe presionar sobre el menú para que se muestran as opciones disponibles.

  • Ahora, pulse la flecha negra a la derecha para abrir el menú.

  • Suavemente, arrastra el cursor sobre cada selección. Fíjese que cada opción será marcada mientras arrastra el curso sobre cada una.

  • Escoge una de las opciones y pulse el ratón.

  • Se puede abrir el menú otra vez para hacer otra selección.
