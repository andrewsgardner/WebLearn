Additional Skills: Drop Down Menus

 

Where would you like to go?
  • The box to the left with the word 'Choose...' in it is a drop down menu.

  • Click the black triangular arrow on the right side of the box to open the menu.

  • Move the cursor over each choice in the menu. Notice that each option will be highlighted while the cursor is over it.

  • Select one of the options by clicking on it.

  • You can open the menu again to select a different option.
